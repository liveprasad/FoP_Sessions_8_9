# Push this project to GitHub

Your local repo is ready and `origin` is set to:

**`git@github.com:pashTheCoder/FoP_Sessions_8_9.git`**

## 1. Create the repository on GitHub

1. Go to **https://github.com/new**
2. **Repository name:** `FoP_Sessions_8_9` (must match exactly)
3. **Visibility:** Private or Public (your choice)
4. **Do not** add a README, .gitignore, or license (you already have them locally)
5. Click **Create repository**

## 2. Push from your machine

In a terminal, from this folder:

```bash
cd /Users/pshetkar/IdeaProjects/Practice/FoP_Sessions_8_9
git push -u origin main
```

(If your project is under `practice` with a lowercase ‘p’, use that path instead.)

Your SSH key is already on GitHub (pashTheCoder), so the push should succeed after the repo exists.
